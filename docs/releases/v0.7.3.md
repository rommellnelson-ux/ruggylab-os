# RuggyLab OS v0.7.3

## Résumé

Cette release stabilise le workflow valeurs critiques et ajoute un cockpit qualité consolidé pour les responsables laboratoire.

## Points fonctionnels

- Rapport valeurs critiques filtrable par examen et unité.
- Synthèse qualité avec hors délai, attentes, examens et unités contributrices.
- Dashboard Qualité centralisant valeurs critiques, TAT, QC et NC/CAPA.
- UAT Playwright isolé par unité unique pour éviter les collisions de données.
- Script de nettoyage UAT local en dry-run par défaut.

## Validation

- Ruff, Bandit et pip-audit passés localement.
- Suite pytest complète passée localement.
- UAT valeurs critiques et contrôle responsive passés localement.

## Notes d'exploitation

- Les scripts UAT créent des patients, échantillons et résultats de test. Les exécuter uniquement sur une base dev ou staging.
- Le warning WebGL TensorFlow vu en Playwright local reste non bloquant et dépend de l'environnement navigateur.
- `v0.7.2` reste publié mais la CI GitHub était rouge sur le formatage Ruff ; `v0.7.3` le remplace comme release propre.
